## 一、模拟说明

### 1.1 为什么选择4月30日作为截止日

2026年5月3日是周日，且处于五一劳动节假期（5月1日-5月5日）中。A股最后一个交易日为**4月30日（周四）**，节后5月6日（周三）恢复交易。

因此本次模拟以**4月30日**为数据截止日——即假设我们在五一假期中打开系统，系统只能看到4月30日及之前的行情数据。

### 1.2 数据穿越控制

| 阶段 | 使用的数据 | 说明 |
|------|-----------|------|
| v7 评分 | 截止日（4/30）及之前 | 基本面 + 知识分 + 历史K线计算的技术指标 |
| 辩论选股 | 截止日之前 | 世界知识库 + 基本面JSON + 技术面评分 |
| 收益验证 | 4/30收盘价 → 6/3收盘价 | **约20个交易日的真实前瞻收益** |

**选股阶段完全看不到4月30日之后的行情**，收益验证阶段才用真实后续数据。

### 1.3 验证周期说明

4月30日后至今（6月3日）实际只有约20个交易日（五一假期5天 + 周末），不到标准的30个交易日。因此本次验证的"30日收益"实际约20个交易日，标注为 `T+~20`。

## 五、关键发现

### 5.1 Judge分高 ≠ 短期收益高

| 排名方式 | Top3 |
|----------|------|
| 按Judge分排 | 东方财富(216) > 金山办公(211) > 迈瑞医疗(203) |
| 按真实收益排 | 新易盛(+49%) > 中际旭创(+49%) > 拓荆科技(+38%) |

Judge分最高的3只股**全部亏损**，而真实收益率最高的3只Judge分反而偏低（53~58分）。这说明辩论裁判倾向于奖励"论据全面"而非"行情爆发力"，AI算力股在Judge分上被低估。

### 5.2 AI算力 = 绝对主线

前5名中4只AI算力相关（光模块×2 + 存储×2），平均收益+37.1%。中际旭创在回测11个窗口中**出现8次**，是本策略最稳定的核心标的。

### 5.3 Bear投降 = 双刃剑

6只触发Bear投降的股票中：3只大涨（平均+39.8%），3只下跌（平均-7.0%）。投降说明做空方确认弱势，但不保证短期一定涨——还要看市场整体环境。

### 5.4 与历史回测一致性

| 回测口径 | 平均收益 | 胜率 |
|----------|----------|------|
| 5/3模拟（本次） | +16.68% | 60% |
| 滚动回测 9窗口均值 | +15.99% | 65.6% |

本次模拟收益与历史回测均值高度一致，验证了策略的稳定性。

## 六、方法论总结

### 6.1 选股流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 白名单    │    │ v7评分    │    │ 四轮辩论   │    │ 30日验证   │    │ 最终推荐   │
│ 3363只   │───→│ Top100   │───→│ 100→50    │───→│ 真实收益   │───→│ TOP10    │
│          │    │ 知识+技术  │    │ →30→20→10 │    │ 排序       │    │          │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### 6.2 辩论机制 v3

- **交互式辩论**: Bull陈述→Bear反驳→Bear陈述→Bull反驳
- **语义+数据双维度反驳**: 基于语义对立词库和同领域数据优势
- **多路径投降机制**: 核心论据压制/连续被反驳/信息严重缺失/权重碾压

### 6.3 无数据穿越保障

所有评分和选股逻辑均只用截止日及之前的数据；收益验证才使用后续实际行情。本报告中的收益数字均为**真实前瞻收益**，不存在未来函数。
